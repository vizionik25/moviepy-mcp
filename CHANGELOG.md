# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-14

### Added
- Initial implementation of MoviePy v2.2.x wrapper.
- FastAPI backend with routers for video, audio, and compositing.
- FastMCP server exposing video editing tools.
- Next.js frontend with LiteLLM and MCP client integration.
- Comprehensive API documentation and README.
- Docker support for both backend and frontend.
- Security layer with path validation for media processing.
- Advanced editing features: scene detection, color effects, time effects, etc.
- LLM Chat endpoint for AI-assisted video editing.
